Учебный проект 2 курса

Инструкция по сборке:
Проект разработан в среде QtCreator, так что для сборки приложения
понадобится версия Qt не ниже 4.0
Сборка осуществляется с помощью системы qmake. 
Важно наличие компилятора gcc на устройстве, т.к. он используется при сборке с qmake

Для сборки нужно открыть консоль в папке с исходными файлами кода (проверьте
наличие .pro файла) и прописать "qmake"

После вывода системы в консоль нужно прописать "nmake"(для
генерации как релизной, так и debug версии) или "nmake release"в случае, 
если нужна только максимально оптимизировання версия приложения

Дождитесь консольного вывода, после чего можно перейти в папку release и прописать
в консоль название сгенерированного exe файла для запуска приложения
(по умолчанию он называется Uchebka_Photoshop) или просто кликнуть на exe файл в 
проводнике.  
В случае, если какой-то из шагов не получился, проверьте наличие путей 
к компиляторам в переменной окружения PATH.

Для того чтобы пересобрать проект, достаточно прописать "nmake clean", после чего
"nmake" с нужными конфигурациями.
